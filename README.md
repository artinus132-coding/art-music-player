# art-music-player
​# 🎵 Artinus Music Player

Artinus Music Player adalah aplikasi pemutar musik berbasis web modern yang memungkinkan pengguna untuk memutar, mengelola, dan mengorganisir koleksi musik mereka langsung dari browser secara gratis.

## 🚀 Fitur Utama
* **Upload Lagu Lokal**: Putar file audio (.mp3, .wav) langsung dari perangkat Anda.
* **Manajemen Playlist**: Buat dan hapus daftar putar (playlist) sesuai keinginan.
* **Kontrol Pemutar**: Fitur Shuffle (acak), Repeat (ulang), dan kontrol volume.
* **Pencarian Cepat**: Cari lagu favorit berdasarkan judul atau artis.
* **Filter Genre**: Kelompokkan musik berdasarkan kategori (Pop, Rock, Jazz, dll).

## 🛠️ Teknologi yang Digunakan
* **HTML5** & **CSS3** (Responsive Design)
* **JavaScript** (Vanilla JS untuk logika pemutar musik)
* **Google Fonts** (Poppins & Inter)

## 🌐 Cara Menggunakan
Cukup buka link website yang tersedia di bagian "About" repository ini, upload file musik Anda, dan mulai mendengarkan!
